# Scrum-ritarit-front-end

Fron-end for Scrum-ritarit project.

## How to use

1. Clone the repository
2. Start from terminal with: npm run dev

### Technologies used
1. React
2. TypeScript
3. Vite
4. Reactstrap
5. Bootsrap
6. Axios

#### Features
1. Search for ticket
2. Mark ticket as used
3. Possibility to undo the ticket marking as used
